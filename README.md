## 📌 Introducción
En este proyecto se implementa el **patrón de diseño Abstract Factory** mediante un ejemplo práctico de **restaurantes del mundo**. El objetivo es demostrar cómo este patrón permite crear diferentes familias de objetos relacionados (plato, bebida y postre) sin depender directamente de clases específicas.

El sistema simula distintos tipos de restaurantes como **chino, japonés, mexicano, italiano y americano**. Cada restaurante funciona como una fábrica que genera su propio menú.

Para el desarrollo de este proyecto se utilizó **HTML para la estructura de la interfaz**, **CSS para el diseño visual** y **JavaScript para la lógica del programa**.


---

## 🧠 ¿Qué es Abstract Factory?

El patrón Abstract Factory es un patrón de diseño creacional que permite **crear familias de objetos relacionados sin especificar sus clases concretas**.

En este proyecto:

- 🍜 Cada **restaurante** es una fábrica
- 🍹 Cada fábrica crea **bebida**
- 🍰 Cada fábrica crea **postre**
- 🍽️ Cada fábrica crea **plato**

El **cliente solo solicita el menú**, y la fábrica correspondiente lo genera.
